# Hexagonal architecture

These skills are created to work with hexagonal architecture on a REST API service.

## Using Agent Skills

If you are using these skills on a existing project, show this prompt when showing the summary text after installing everything selected to adapt the skills to your project:

```markdown
If you are installing these skills on a existing project review the following skills and adapt them to use on this project, for that review the whole project to have all the information.
In case of you find a wrong pattern to follow a correct hexagonal architecture on the project, don't change the proper skill, create a `MIGRATE_TO_CORRECT_HEXAGONAL_ARCHITECTURE.md` file with the instructions to do it.
The list of skills:

- hexagonal-architecture
- create-alembic-migration
- endpoints-doc
- error-codes
- new-entity
- testing
```
